# React Search Songs Application

This is a React project created to search, sort, and filter through a list of songs.

The project is deployed with gh-pages here: [Search Songs App](https://projectym.github.io/Search-Songs-App/).


## What This App Does:

This app uses fetch to grap JSON data from a file on GitHub to then display.<br/>

It then uses the "useContext" hook to take the state and distribute it to the desired components.<br/>

The search function finds songs that match each input (separated by a space ' '), inclusively (A&B)<br/>

One can also use the sort and filter settings to change what is displayed<br/>

In the menu, there is also a "Dark Mode" option!<br/>
